🌍 Environment Saver – OpenGL 2D Game


📌 Overview

Environment Saver is an interactive 2D OpenGL-based game designed to promote environmental awareness through engaging gameplay and real-time graphics. The project simulates a dynamic city environment where users perform tasks to restore ecological balance and prevent environmental destruction.

🎮 Key Features

✨ Dynamic Environment

Day–night cycle 🌇
Weather system (rain & lightning) 🌧️⚡

🌱 Gameplay Mechanics

Tree plantation system
River cleaning system
Pollution control (factory smoke)
Park construction

🚗 Interactive Elements

Moving vehicles (car & truck)
Player movement with keyboard

🔥 Advanced Visual Effects

Particle systems (smoke, fire, explosion, O₂)
Cinematic explosion with sound 💥
Camera shake & slow-motion effects

🎯 Game System

Level-based progression
Timer-based challenge
🕹️ Controls
Key	Action
⬅️ ➡️ ⬆️ ⬇️	Move Player
T	Plant Tree
C	Clean River
F	Stop Factory Smoke
B	Build Park
R	Toggle Rain


🧠 Objective
Complete all environmental tasks within the time limit to save the city.

✅ Success: Oxygen (O₂) is generated, indicating a healthy environment
❌ Failure: The city is destroyed with cinematic explosion effects
🛠️ Tech Stack
Language: C++
Graphics: OpenGL (GLUT)
Audio: Windows Multimedia API (PlaySound)
🚀 How to Run
# Compile (example using g++)
g++ main.cpp -o game -lglut -lGL -lGLU -lwinmm
Install OpenGL (GLUT)
Place required files:
bird.wav
Run the executable
