🌍 Environment Saver – OpenGL 2D Game
📌 Overview

Environment Saver is an interactive 2D game developed using OpenGL that promotes environmental awareness through real-time graphics and gameplay. The game simulates a dynamic city where players perform tasks to restore and protect the environment.

🎮 Features
🌇 Dynamic day-night cycle
🌧️ Weather effects (rain & lightning)
🚗 Moving vehicles (car & truck)
🌳 Tree plantation system
🌊 River cleaning mechanics
🏭 Pollution control (factory smoke)
🏞️ Park building system
💨 Particle effects (smoke, fire, explosion, O₂ bubbles)
💥 Cinematic explosion with sound and screen shake
🎯 Level-based gameplay with timer
🕹️ Controls

Key	Action
⬅️ ➡️ ⬆️ ⬇️	Move player
T	Plant tree
C	Clean river
F	Stop factory smoke
B	Build park
R	Toggle rain
🧠 Game Objective

Complete all environmental tasks within the given time to save the city.

If successful → 🌱 Oxygen (O₂) is produced
If failed → 💥 City is destroyed with explosion effects
🛠️ Technologies Used
C++
OpenGL (GLUT)
Windows Multimedia API (PlaySound)

🚀 How to Run
Install OpenGL (GLUT)
Compile using CodeBlocks / Dev-C++ / g++
Place sound files (bird.wav, explosion.wav) in project folder
Run the program
📸 Visual Effects
Real-time animation
Particle systems for smoke, fire, and oxygen
Camera shake & cinematic explosion
Smooth object movement and interaction

🎯 Learning Outcomes
2D transformations and animation
Real-time rendering
Game logic and event handling
Particle system implementation
